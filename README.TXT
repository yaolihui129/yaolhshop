[系统说明]
《细说PHP》第三版中的BroSHOP是一套中小型电子商城系统、是商品发布、在线交易的系统平台.
 它操作简单,易于维护,更采用了WEB开发领先技术,目前版本为1.0Beta版. 
 仅供《细说PHP》读者及兄弟连IT教育学员学习交流使用。

[系统需求]
  Apache2.2版本以上，加载PHP动态模块
  PHP5.2版本以上,一定要支持GD库及PDO等扩展
  MYSQL5.5版本以上
  
[安装方法1]
  1. 将软件包解压后上传到Web服务器的文档根目录或子目录下
  2. 在浏览器中正确的输入URL（http://域名/项目目录/install/index.php）,自动进入安装页面。
  3. 按照提示信息安装即可，安装前要先创建好数据库，在安装提示下需要将一些目录和文件设置成能让PHP可以写的权限。
  
  安装过程中需要修改配置文件,一旦出现问题,
       如果runtime目录下有install.lock文件存在，可以删除该文件后重新安装即可.
  注意:
     1.安装成功可以请将install目录删除,以防安全隐患.
	 
[安装方法2] 
	《细说PHP》第三版光盘中，还为本系统提供了一套带有演示数据的版本。
	  1. 只要将数据库目录（dbshop）放到你的数据库目录下（MySQL数据目录下）。
	  2. 将所有程序放在自己的Web目录下。
	  3. 修改主配置文件config.inc.php 正确配置自己的数据库信息
	  
	  访问admin.php进入后台， 默认管理用户名和密码都是admin
  
[使用方法]
  1. 安装后在浏览器中运行安装路径下的/admin.php文件进入后台。
  2. 进入后台管理入口，在登录界面中输入安装时设定的管理员用户名和密码。
  
[系统特点]
	使用流行脚本语言PHP编写，搭配性能稳定的MySQL数据库. 
	系统可以跨平台，运行在Windows、Linux等操作系统中. 
	使用最新的ck编辑器，发布文件排版像使用WORD一样简单. 
	使用DIV+CSS技术布局,遵循WEB标准,兼容各种浏览器. 
	采用Smarty模板引擎,页面高速缓存技术. 
	采用完全的MVC模式,并使用自定义框架. 
	采用完全的PHP5面向对象设计. 
	支持无限级的分类与子分类. 
	系统采用超经量级PHP框架BroPHP2.0实现.  
	代码经典，极具有参考和学习价值。
